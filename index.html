Technical Support

This page provides technical support information for the game Civilization Directive.

Civilization Directive is an independent strategy simulation game developed by the independent developer behind Hawting Studio.

Basic Information

• Platform: iOS / iPadOS
• Minimum system requirement: iOS 15 or later
• Game type: Offline, standalone application

Common Issues

• The game takes longer to load on first launch
This is normal, especially on older devices. Subsequent launches will be faster.

• Performance may vary on older devices
For best performance, ensure sufficient free memory and close background apps.

• Progress and settings
Game progress is stored locally on the device.

Data & Privacy

Civilization Directive does not collect, store, or transmit personal data. No accounts, analytics services, advertising SDKs, or third-party tracking tools are used.

Platform-level services (such as operating system features) may process data independently, in accordance with their own privacy policies.
